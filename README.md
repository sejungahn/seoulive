# Project - Seoulive
스프링 부트 + JSP 일정, 코스 제공 사이트


## 🖥️ 프로젝트 소개
트립소다 및 트리플을 참고하여 만든 여행 일정 제공 사이트입니다.
<br>

## 🕰️ 개발 기간
2023.04. - 2023.05

### ⚙️ 개발 환경
- `Java 17`
- `JDK 17`
- **IDE** : IntelliJ IDEA
- **Framework** : Spring Framework
- **Database** : Oracle 19c
- **ORM** : Mybatis
